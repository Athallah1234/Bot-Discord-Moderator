# Security Policy

## Supported Versions

Hanya versi terbaru yang mendapatkan pembaruan keamanan.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

Jika Anda menemukan celah keamanan, **mohon jangan buka Issue publik**. Sebagai gantinya, silakan kirim laporan ke email kami di `security@yourdomain.com` atau hubungi Admin melalui Discord.

Kami akan merespon dalam waktu 48 jam dan memberikan pembaruan segera setelah masalah teratasi.
