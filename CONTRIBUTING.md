# Contributing to Ultimate Discord Suite

Terima kasih telah tertarik untuk berkontribusi pada proyek ini! Kami sangat menghargai bantuan Anda untuk membuat bot ini menjadi lebih baik.

## Cara Berkontribusi

### 1. Melaporkan Bug
Jika Anda menemukan bug, silakan buka **Issue** baru dengan label `bug`. Jelaskan langkah-langkah untuk mereproduksi bug tersebut.

### 2. Menyarankan Fitur
Punya ide keren? Buka **Issue** baru dengan label `enhancement`.

### 3. Mengirim Pull Request
1. Fork repositori ini.
2. Buat branch baru (`git checkout -b feature/FiturKeren`).
3. Lakukan perubahan dan commit (`git commit -m 'Menambahkan fitur keren'`).
4. Push ke branch Anda (`git push origin feature/FiturKeren`).
5. Buka Pull Request ke branch `main`.

## Standar Kode
- Gunakan ES Modules (`import`/`export`).
- Pastikan kode Anda rapi dan memiliki komentar jika diperlukan.
- Dokumentasikan perintah baru di `README.md`.

Terima kasih atas kontribusi Anda!
